## Create Step
```$xslt
1. 安装
npm install -g vue-cli
npm install -g webpack

2. 创建工程
vue init webpack 02_demo

3. 创建该目录下的工程结构
3.1 npm install -save-dev element-ui
3.2 编写页面和路由

```

## Build Setup

```$xslt
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
